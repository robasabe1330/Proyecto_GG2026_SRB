# Rio Bita Flood Mapping (2015–2024)

This repository contains Google Earth Engine scripts used to map seasonal
flood dynamics in the Rio Bita basin (Colombia).

## Scripts

Script 1 — Seasonal composites generation  
Script 2 — Flood mapping and frequency calculation  

## Requirements

Google Earth Engine account

## Asset configuration

Replace the asset path:

var ASSET_PATH = 'projects/user/assets/';

## Method

SWIR threshold method following:

Wolski et al. (2017)  
Inman & Lyons (2020)

## Study area

Rio Bita basin, Colombia
